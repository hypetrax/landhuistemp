# Landhuis Installatietechniek Temporary Website

Temporary Vite/React landing page for Landhuis Installatietechniek while the full website is being prepared.

## Development

```bash
npm install
npm run dev
```

## Checks

```bash
npm run lint
npm run build
```

## Project Notes

- Main page content is in `src/App.tsx`.
- Styling is split between `src/index.css` and `src/App.css`.
- Public assets used by the page live in `public/`.
- Production output is generated in `dist/` and should not be edited directly.
